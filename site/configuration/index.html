Konfiguration

Die Konfiguration besteht aus zwei Teilen:

  1. Access Token
  2. API Base URL

1. Access Token

Damit die ChurchTools Utilities funktioneren, muss auf dem Rechner, auf dem die Utilities laufen, die Umgebungsvariable CHURCHTOOLS_TOKEN mit dem ChurchTools Access Token angelegt sein.

ChurchTools Access Token besorgen

In ChurchTools Web bekommst du das Token in Personen & Gruppen > Personenliste > „Person A“ > Berechtigungen > Login-Token

Umgebungsvariable setzen

Um die Umgebungsvariable CHURCHTOOLS_TOKEN für den aktuellen Benutzer in Windows zu setzen, führe die folgenden Schritte aus:

  1. Öffne die Umgebungsvariablen-Einstellungen:
  2. Drücke Win + R, gib sysdm.cpl ein und drücke Enter.

  3. Gehe zum Tab Erweitert und klicke auf Umgebungsvariablen.

  4. Neue Benutzer-Umgebungsvariable hinzufügen:

  5. Klicke unter Benutzervariablen für frank auf Neu.
  6. Setze den Namen der Variablen auf CHURCHTOOLS_TOKEN.

  7. Setze den Wert der Variablen auf deinen Token.

  8. Speichern und Schließen:

  9. Klicke auf OK, um die Änderungen zu speichern.

  10. Schließe alle Dialoge.

  11. PowerShell neu starten: Starte die PowerShell neu, damit die Änderungen wirksam werden.

  12. Überprüfen, ob die Variable gesetzt ist Führe in der PowerShell den folgenden Befehl aus, um zu überprüfen, ob die Umgebungsvariable korrekt gesetzt wurde:

echo $env:CHURCHTOOLS_TOKEN

Wenn der Token angezeigt wird, ist die Umgebungsvariable korrekt gesetzt. Du kannst jetzt dein Python-Skript ausführen. Wenn der Token angezeigt wird, ist die Umgebungsvariable korrekt gesetzt. Du kannst jetzt dein Python-Skript ausführen.

Um eine Umgebungsvariable permanent für einen Benutzer auf einem Linux System oder einem einem Raspberry Pi zu setzen, kannst du die .bashrc-Datei des Benutzers bearbeiten. Hier sind die Schritte:

  1. Öffne die .bashrc-Datei und füge die Umgebungsvariable hinzu Füge am Ende der Datei die folgende Zeile hinzu und speichere die Datei:
export CHURCHTOOLS_TOKEN="<token value here>"
  1. Aktualisiere die Umgebungsvariablen Lade die .bashrc-Datei neu, damit die Änderungen wirksam werden:
source ~/.bashrc
  1. Überprüfe die Umgebungsvariable Prüfe, ob die Umgebungsvariable korrekt gesetzt wurde:
echo $CHURCHTOOLS_TOKEN

Wenn der Token angezeigt wird, ist die Umgebungsvariable korrekt gesetzt.

2. API Base URL

Setze die BASE URL in der Datei config.py auf den Wert für deine Kirchengemeinde:

# Base URL der ChurchTools Instanz API
BASE_URL = "https://volksdorf.church.tools/api"
